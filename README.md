# Smart Image Captioning Assistant for People with Visual Challenges

## Overview

An AI-powered accessibility assistant designed to help visually impaired users understand digital images through:

- Image Captioning
- Arabic Translation
- Text-to-Speech
- Accessible User Interface

The system generates image descriptions, translates them into Arabic, and converts them into speech to improve accessibility and user independence.

---

## Features

- Upload images through a simple interface
- Generate descriptive image captions using BLIP Large
- Translate captions into Arabic
- Convert Arabic captions into speech
- Accessibility-focused design for visually impaired users

---

## Technologies Used

- Python
- PyTorch
- Hugging Face Transformers
- BLIP Large
- Streamlit
- Deep Translator
- gTTS
- Pillow

---

## Project Workflow

Image Upload
→ Caption Generation
→ Arabic Translation
→ Text-to-Speech
→ Audio Output
